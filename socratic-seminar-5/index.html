Temes

  • CTV-CSFS - Un grup de desenvolupadors han firmat una carta demanant la revisió i activació dels soft-forks CTV i CSFS (BIPS 119 i 348 respectivament).
  • Dust Expiry: Clean the UTXO set from spam -- RobinLinus proposa un soft-fork per eliminar el dust que no sera gastat del UTXO set.
  • Emulating OP_RAND -- El llenguatge d'scripting the Bitcoin no permet (de manera directa) introduir aleatorietat en les condicions de despesa. Oleksandr Kurbatov proposa un protocol interactiu entre dues parts que permet incorporar aquesta aleatorietat de forma controlada, emulant el que ell n'anomena l'OP_RAND opcode. OPTECH, Format Paper.
